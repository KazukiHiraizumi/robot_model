# ROS 3D Robot Model example
## Run
1. Run ros core
~~~
roscore
~~~

2. Launch
~~~
roslaunch test.launch
~~~

3. Take a look
~~~
rviz
~~~

## URDF modelling notice
1. Origin

The origin of the link(or arm) is joint to the parental link. The joint point to child link is represented by the origin of its joint
